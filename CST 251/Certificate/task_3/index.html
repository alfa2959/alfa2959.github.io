Introduction

HTML (Hyper Text Markup Language) is the standard for creating web pages. Tim Berners-Lee created the first version of HTML in the year 1993. Since then, there have been many versions of HTML (the most recent being HTML5). The uses of HTML include web development, internet navigation, and web documentation.

How HTML Works

HTML essentially tells the internet browser what to display. By inputting code into an HTML document, the web browser is able to read the file to display its contents.

HTML Basics

HTML pages have HTML elements, that consist of a set of tags and attributes. HTML elements are what build the web page.

The following are some of the most common tags:

  • The <a> tag creates a link.

  • The <br> tag creates a line break.

  • The <h1> tag is used for titles.

  • The <p> tag is used for paragraphs.

  • The <div> tag is used as a divison/part of an HTML document.
